# 洛克王国世界孵蛋查询

基于 **zhenxun_bot** 架构编写的插件。在群聊内根据宠物蛋径、蛋重，查询可能孵出的洛克王国宠物。

## 功能

在群聊内输入宠物的蛋径和蛋重，根据公开API自动提交返回可能孵出的宠物，并生成一张包含以下信息的表格图片：

- 序号
- 宠物头像
- 宠物名称（蛋形态会以 🥚 标记）
- 匹配度与置信度（高/中/低，带颜色标识）
- 属性与进化链

## 指令

```
查蛋 蛋径 蛋重
```
- 执行查蛋

```
查蛋
```
- 不带参数时返回插件帮助
